pyshare is a terminal based file uploader.

Prerequisites & requirements
----------------------------

- Python 3.7.0 or higher
- pip 18.1 (Python 3.7.0) or higher

Installation
------------

- Initial install::

    pip install pyshare
    Copy .pyshare-template.json to `~/.pyshare.json`

- Updating::

    pip install --upgrade pyshare

Usage
-----

 ``pyshare [args]``

 -h            Help dialog.
 -v            Version.
